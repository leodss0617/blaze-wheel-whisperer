# Blaze Analyzer

Bot de análise para o jogo Double da Blaze com coleta de dados em tempo real.

## Funcionalidades

- Conexão com API da Blaze via Edge Function proxy
- Histórico de rodadas em tempo real
- Estatísticas detalhadas (percentuais, sequências, streaks)
- Sistema de sinais com algoritmo de previsão baseado em padrões
- Gráfico de tendência

## Tecnologias

- React + TypeScript + Vite
- Tailwind CSS
- Lovable Cloud (Edge Functions)
- Recharts para gráficos
